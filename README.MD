<div align="center">

KinAi-ScholarKit is an interactive web application built with Streamlit, designed to generate educational activities aligned with the **Victorian Curriculum Fâ€“10 Version 2.0**. It leverages a Retrieval-Augmented Generation (RAG) system, powered by local Large Language Models (LLMs) via Ollama, to provide tailored lessons, stories, and activities for various learning areas and year levels (Foundation to Year 10). The app uses a knowledge graph, FAISS for vector search, and LLMs to create engaging content, downloadable as PDF.
